売上管理ツール Ver18.5 OpenAI本読取版

変更:
- 画像のOpenAI本読取を追加
- AI候補の編集欄を追加
- 編集候補保存を追加
- 仕入へ登録を追加
- 売上へ登録を追加

使い方:
1. GitHubへ上書きアップロード
2. Vercelの自動更新を待つ
3. 公開URLを開く
4. OpenAI APIキーを保存
5. 画像を証憑登録
6. OpenAI本読取
7. 編集して売上/仕入へ登録

注意:
- Ver18.5は画像本読取対応です。
- PDF本読取は次版で対応予定です。
- 公開URLでAPIキーをブラウザ保存する方式は簡易版です。本番はサーバー側API化推奨です。
